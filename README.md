## ğŸš€ Getting Started

### Prerequisites

- **Node.js**: >= 16.x
- **npm**: >= 7.x
- **MongoDB** (for the backend)

### Installation

Follow these steps to set up the project:

#### Clone the repository:

```bash
git clone https://github.com/MetastartTech/com.genmanai.git
cd genman-ai
```

#### Install dependencies:

For the frontend:

```
cd frontend
npm install

```

For the backend:

```
cd backend
npm install

```

#### Run the application:

Start the frontend:

```
cd frontend
npm run dev

```

Start the backend:

```
cd backend
npm run dev

```
